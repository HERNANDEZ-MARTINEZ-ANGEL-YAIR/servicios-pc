# Sistema de Gestión de Servicios PC

Sistema web para administración de servicios de archivos, software y soporte técnico.

## Características
- Panel de administración
- Gestión de usuarios
- Control de servicios
- Sistema de login seguro

## Tecnologías
- PHP
- MySQL
- HTML5/CSS3
- JavaScript

## Instalación Local
1. Clonar repositorio
2. Importar base de datos
3. Configurar conexión en `public/conexion.php`

## Despliegue en Render
Este proyecto está configurado para desplegarse automáticamente en Render.
